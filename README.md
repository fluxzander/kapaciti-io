## The 9 Agents

Every agent runs on schedule. Specialized prompt, specialized skills, specialized output.

### NEXUS — Command Layer
Routes every incoming task to the right specialist automatically. Parses intent, enriches context, dispatches to the correct agent, reports back. The orchestration layer that makes the other eight work together.

### PULSE — Market Intelligence
Publishes a daily market briefing before trading opens. Macro, sentiment, economic calendar, geopolitics. Written in your voice. Posted to the right channel. Every trading day, 09:00.

### COACH — Client Coaching Engine
Responds to your clients in your exact voice. Reads their trade journal, identifies FOMO, oversizing, revenge trading, psychology patterns. Drafts coaching messages for your approval. Never generic. Never wrong tone.

### FORGE — Revenue & Pipeline
Finds prospects, enriches with email and LinkedIn data, runs multi-touch outreach sequences, tracks responses. Integrates with Stripe for invoicing. Runs Monday–Friday, silent otherwise.

### CRAFT — Content Engine
Turns any piece of content — a tweet, a voice memo, a call transcript — into threads, newsletters, video scripts and email sequences. Repurposes automatically. Schedules via Typefully.

### BUILD — Code & Infrastructure
Writes code, manages Supabase databases, deploys APIs, runs Playwright tests, creates GitHub PRs. Handles full-stack: schema migration to production deploy.

### REACH — Social Execution
Posts to TikTok, Instagram and X automatically. Follows your content calendar. Adapts copy per platform. Tracks engagement signals and adjusts cadence.

### RADAR — Signal Detection
Monitors Reddit, X, TradingView and HackerNews for high-signal mentions, setup alerts, competitor moves and trend shifts. Filters noise. Surfaces only what matters.

### VOICE — Brand Copywriting
Writes email sequences, ad copy, landing pages and sales scripts in your brand tone. Trained on your existing content. Sounds like you. Not like an AI.

```env
# .env — add keys for providers you want, leave others blank

ANTHROPIC_API_KEY=sk-ant-...       # Claude (default)
OPENAI_API_KEY=sk-proj-...         # GPT-4o, Whisper, DALL-E
GOOGLE_API_KEY=AIza...             # Gemini, Veo3, Imagen
MOONSHOT_API_KEY=sk-...            # Kimi — cheapest long-context
DEEPSEEK_API_KEY=sk-...            # DeepSeek R1
GROQ_API_KEY=gsk_...               # Groq — fastest
MISTRAL_API_KEY=...                # Mistral
XAI_API_KEY=xai-...                # Grok
OLLAMA_HOST=http://localhost:11434  # Local Ollama — no key needed

# Global default
MODEL=anthropic/claude-sonnet-4-6

# Per-agent overrides — mix freely
PULSE_MODEL=openai/gpt-4o
FORGE_MODEL=moonshot/moonshot-v1-128k    # Cheapest for high-volume
BUILD_MODEL=groq/llama-3.3-70b          # Fastest for code
COACH_MODEL=ollama/llama3.3:70b         # Local — client data stays offline
RADAR_MODEL=groq/llama-3.1-8b-instant   # Sub-50ms for real-time signals
```

**Cost optimization:** Running all 9 agents with cheap providers (Kimi + Groq + Ollama + DeepSeek) costs ~$5–15/month. Full setup details in [docs/AI-ENGINES.md](docs/AI-ENGINES.md).
