// faq

Common questions

Do I get access to your personal data or accounts?

Never. The private repo contains generic code only. You connect your own Stripe, Discord, Cal.com, and OpenAI keys. Your data stays completely separate.

What do I need to run this?

A Mac or Linux machine with Node.js installed. OpenClaw or Claude Code for the agent skills. API keys for Stripe, OpenAI, and optionally Cal.com and Discord. That's it.

How long does setup take?

The self-install ($99) takes about 30 minutes. Clone the repo, run the setup script, enter your API keys, and you're live. The done-for-you ($1000) means we handle everything.

What are the 70+ skills exactly?

OpenClaw skills covering: revenue tracking, agent orchestration, social media automation (X, Instagram, TikTok), lead generation, content repurposing, SEO, email campaigns, Stripe integration, Supabase, Notion, cognitive memory, churn prediction, calendar prep, and much more.

Is this a SaaS? Do I pay monthly?

No. One-time payment. You own the code. You host it yourself. No subscriptions, no recurring fees. You pay for your own API usage (OpenAI, ElevenLabs, etc.) directly.
