+

A Mac or Linux machine with Node.js. OpenClaw or Claude Code for agent skills. API keys for Stripe, OpenAI, and optionally Cal.com and Discord. Nothing else.

Self-install ($99) takes about 30 minutes. Clone the repo, run the setup script, enter your API keys, and you're live. Done-for-you ($1000) means we handle everything.
